# Telecom AI Analytics Copilot

An end-to-end Data Engineering and AI Analytics project.

## Features

- Airflow ETL
- Data Quality Checks
- DuckDB Warehouse
- LangGraph AI Agent
- Ollama LLM
- SQL Generation
- Automated Insights
- Interactive Dashboard

## Architecture

Telecom Data
    ↓
Airflow
    ↓
Data Quality
    ↓
DuckDB
    ↓
LangGraph Agent
    ↓
Ollama
    ↓
SQL Generation
    ↓
DuckDB Query
    ↓
Charts + Insights
